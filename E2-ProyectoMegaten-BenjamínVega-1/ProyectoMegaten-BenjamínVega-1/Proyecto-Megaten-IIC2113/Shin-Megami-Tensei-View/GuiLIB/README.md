# GuiLIB (no se versiona)

Coloca aquí la carpeta **GuiLIB** entregada con la cátedra. Debe contener al menos el archivo `SMTGUI.dll` y los recursos que requiere la vista gráfica.

La aplicación la buscará en este directorio relativo al repositorio. Alternativamente puedes definir las variables de entorno `SMTGUI_LIB` (ruta completa al DLL) o `SMTGUI_DIR` (directorio que contiene el DLL) antes de ejecutar el juego.

> **Importante:** Esta carpeta no se versiona en Git. Agrégala manualmente cuando ejecutes el proyecto en tu máquina local.
